# EPSI HL - App de Remisiones y Turnos

Esta carpeta contiene la base del proyecto para los módulos solicitados:

- Remisiones en PDF con envío por email e historial.
- Gestión de turnos con calendario y notificaciones WhatsApp.
- Gestión de usuarios y roles con JWT.
- BI con métricas y gráficas.

## Estructura propuesta

- `backend/` API REST (Node.js + Express).
- `frontend/` UI web (React + Vite).
- `docs/` guía paso a paso.
